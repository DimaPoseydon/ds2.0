## Запуск приложения

Для запуска приложения необходимо установить все зависимости из файла `requirements.txt`.

### Установка зависимостей

Выполните в терминале следующую команду:

```bash
python -m pip install -r requirements.txt
```
## Запуск сервера и клиента

### Сервер
Для запуска сервера запустите батник сервера.

### Клиент
Для запуска клиента запустите батник клиента.

### IP-адрес
В роли IP-адреса будет выступать **IPv4 адрес компьютера-сервера**.
